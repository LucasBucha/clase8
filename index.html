Cascading Style Sheets (CSS)

CSS es un lenguaje para especificar cómo se presentan los documentos a los usuarios: cómo se diseñan, distribuyen, etc. Se puede utilizar para un estilo de texto de documentos muy básico, por ejemplo, cambiar el color y el tamaño de los encabezados y enlaces. Se puede utilizar para crear un diseño, por ejemplo, convertir una sola columna de texto en un diseño con un área de contenido principal y una barra lateral para la información relacionada. Incluso se puede utilizar para efectos como animación .

CSS es un lenguaje basado en reglas, definimos reglas que especifican grupos de estilos que deben aplicarse a elementos particulares o grupos de elementos en nuestros sitios. Por ejemplo, "Quiero que el encabezado principal de mi página se muestre como texto rojo grande".
